## Cara Install Di Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> git clone https://github.com/adulalhy/SZ-SELF.git
> cd SZ-SELF
> npm i
> node run
> scan QR and Enjoyy
```

## Cara Install Di Windows
* [`Download Node JS`](https://nodejs.org/en/download/)
```cmd
> git clone https://github.com/adulalhy/SZ-SELF.git
> cd SZ-SELF
> npm i
> node run
> scan QR and Enjoyy
```
